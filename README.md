# Dark Folio Press — Free Widget Bundle
### Embed codes for site owners

Copy one of the snippets below into a **Custom HTML / Embed / iframe** element on
Wix, Squarespace, WordPress, or any site builder. Each widget refreshes daily,
carries Dark Folio Press branding, and links back to your 3-Day Horoscopes page.

---

## 🌙 Observatory Widget
Shows: Moon · Element · Atmosphere · Favourable Activities · Correspondence

```html
<iframe
  src="https://slaing77.github.io/3day-scopes/widgets/observatory.html"
  width="360"
  height="420"
  style="border:none; max-width:100%;"
  loading="lazy"
```

## Sizing notes
Each widget is designed for a **360px-wide sidebar-style placement**. On Wix,
set the embed element's width to match (or leave "responsive" on if Wix
offers it) — the widget itself has `max-width` built in so it won't overflow
on mobile.
